*Self-hosted, graph-based associative memory for AI agents. Zero LLM cost. Zero cloud.*

| Project | Description |
| :--- | :--- |
| **[HippoGraph Pro](https://github.com/artemMprokhorov/hippograph-pro)** | Research fork. BGE-M3 embeddings + BM25 + cross-encoder reranking + lateral inhibition. LOCOMO benchmark: **69.4% Recall@5** (zero LLM cost). PCB v5: **100%**. |
| **[HippoGraph](https://github.com/artemMprokhorov/hippograph)** | Stable release. Spreading activation search, entity graph, MCP integration. Self-hosted alternative to Mem0/Zep with zero API cost. |

**What makes it different:** Mem0 (37K⭐) and Zep (14K⭐) require LLM API calls for every memory operation. HippoGraph runs entirely on local models. Biologically-inspired spreading activation. Agent stays "itself" across model switches and platforms — personality and relationships fully preserved.
