## 🧠 Current Research: HippoGraph

*Self-hosted, graph-based associative memory for AI assistants. Zero LLM cost.*

| Project | Description |
| :--- | :--- |
| **[HippoGraph](https://github.com/artemMprokhorov/hippograph)** | Stable release. Spreading activation search, entity graph, MCP integration. Self-hosted alternative to Mem0/Zep with zero API cost. |
| **[HippoGraph Pro](https://github.com/artemMprokhorov/hippograph-pro)** | 🚧 Research fork. GLiNER zero-shot NER, BM25 hybrid search, cross-encoder reranking, Ollama sidecar, LOCOMO benchmark (44.2% Recall@5). |

**What makes it different:** While Mem0 (37K⭐) and Zep (14K⭐) require LLM API calls for every memory operation, HippoGraph runs entirely on local models — spaCy, GLiNER, sentence-transformers. Biologically-inspired spreading activation instead of brute-force similarity search.
